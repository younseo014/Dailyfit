# [PRD v2-A] 데일리핏 (DailyFit) — 자영업자 특화 업무 보조 × 인지 건강 스크리닝 앱

> **방향성 요약**: 자영업자가 매일 쓰는 "업무 도구"로 진입하여, 사용 데이터 기반으로 인지 저하를 2단계로 스크리닝한다. 금전적 리워드로 사용 동기를 만들고, 스마트워치를 통해 헬스케어 기능을 확장한다.

## 1. 제품 개요

- **제품명**: 데일리핏 (DailyFit)
- **한 줄 설명**: 자영업자를 위한 발주·메모 관리 앱이자, 사용 패턴을 통해 인지 건강을 조용히 모니터링하는 웰니스 비서.
- **핵심 가치 제안**: "쓸수록 돈이 쌓이는 업무 도구" — 금전적 인센티브로 자발적 사용을 유도하고, 쌓인 데이터로 인지 건강을 추적한다.
- **타깃 플랫폼**: iOS / Android 네이티브 앱 + 갤럭시워치 / 애플워치 연동

---

## 2. 타깃 사용자 & 페르소나

### 페르소나 A — 52세 편의점 가맹점주 (남)
- **상황**: 매일 발주 품목을 수기로 메모하거나 엑셀에 정리하는데, 가끔 같은 품목을 이중 발주하거나 유통기한 확인을 빠뜨림. 스마트폰은 익숙하지만 복잡한 앱은 거부감이 있음.
- **욕구**: 발주·메모를 빠르게 처리하고 싶고, "이거 쓰면 뭔가 이득이 있어야 쓰지"라는 실리 마인드.
- **잠재 불안**: 최근 단골 손님 이름을 자꾸 잊거나, 방금 발주한 항목을 기억 못하는 일이 잦아졌지만 병원 가기는 꺼려짐.

### 페르소나 B — 48세 미용실 원장 (여)
- **상황**: 예약 관리, 약품 재고, 직원 스케줄을 머릿속으로만 관리하다가 더블 부킹 실수가 반복됨.
- **욕구**: 간단하게 말하거나 입력하면 정리되는 도구. 사용할 때마다 소액이라도 적립되면 더 열심히 쓸 것 같음.

## 3. 핵심 기능 상세

### ① 발주·업무 메모 관리 (핵심 일상 기능)
- **발주 템플릿**: 업종별(카페, 편의점, 미용실 등) 사전 설정 템플릿 제공. 품목명·수량·거래처를 텍스트 또는 시각적 체크리스트로 기록.
- **재고 알림**: 발주 주기를 학습하여 "○○ 재고가 평소보다 늦게 발주되고 있어요" 알림.
- **업무 메모**: 자유 형식 텍스트 메모. 태그(#손님, #거래처, #할일) 자동 분류.
- **음성 입력**: 선택적 보조 기능. 마이크 버튼을 길게 눌러야만 활성화.

### ② 포인트 리워드 시스템
- 발주 기록 1건 완료 시: **2포인트** 적립
- 업무 메모 작성 시: **1포인트** 적립
- 7일 연속 사용 달성 시: **50포인트** 보너스
- 적립 포인트는 **네이버페이 포인트 / 카카오페이 캐시**로 전환 가능 (최소 전환 단위: 1,000포인트 = 1,000원)
- 월 최대 적립 상한: **3,000포인트** (과도한 어뷰징 방지)

### ③ 1단계 인지 패턴 스크리닝 (백그라운드)
앱을 사용하면서 자연스럽게 수집되는 텍스트 데이터에서 아래 지표를 분석:

| 분석 지표 | 예시 | 가중치 |
|---|---|---|
| 지시 대명사 과다 사용 | "저거 두 개", "그거 있잖아요" | +10점/건 |
| 명칭 망각 지연 표현 | "어... 이름이 뭐더라", "그 브랜드 있잖아" | +25점/건 |
| 발주 이중 입력 | 동일 품목을 24시간 내 중복 등록 | +15점/건 |
| 기록 일관성 저하 | 평소 기록 패턴 대비 문장 완성도 감소 | +20점/건 |

- **인지 피로 점수(100점 만점)**: 최근 30일 평균이 40점 이상이면 2단계 진입 트리거.
- 사용자에게는 "두뇌 활동 지수"라는 긍정 프레이밍으로 표시.

### ④ 2단계 인지 테스트 (명시적 진단 도구)
- **진입 트리거**: 1단계에서 기준치 초과 또는 사용자의 자발적 선택.
- **테스트 구성**:
  - 사진-단어 매칭: 상품 사진 12장을 보여주고 5분 뒤 이름 맞추기
  - 발주 순서 기억: 아까 입력한 발주 품목 순서 회상
  - 숫자 역순 입력: 불러주는 숫자를 거꾸로 입력 (표준 인지 검사 기반)
- **소요 시간**: 약 5분
- **결과 제공**: "지금 바로 병원 예약" 버튼 + 가까운 치매안심센터 위치 안내 자동 연결.

### ⑤ 스마트워치 연동 헬스케어
- **갤럭시워치 / 애플워치** API 연동.
- 심박수·수면 패턴·활동량 데이터를 앱으로 수신.
- **활용 방식**:
  - 심박수가 안정적이고 활동량이 낮은 "여가 시간" 감지 시 → 1~2분짜리 인지 미니 퀴즈 알림 발송.
  - 수면 시간이 6시간 미만인 날이 3일 이상 지속되면 "수면 부족이 기억력에 영향을 줄 수 있어요" 알림.
  - 워치 화면에서 바로 발주 메모 빠른 등록 가능 (Wear OS / watchOS 컴패니언 앱).

## 4. 온보딩 — 점진적 고지 구조 (기만 방지)

```
[Step 1] 가입
  → "자영업자를 위한 발주·업무 메모 앱입니다. 쓸수록 포인트가 쌓여요."
  → 업종 선택 / 기본 정보 입력

[Step 2] 첫 사용 후 3일 뒤 자동 팝업
  → "데일리핏은 업무 기록을 분석해 당신의 두뇌 활동 패턴도 함께 관찰해요.
     이 기능을 함께 사용하시겠어요? (설정에서 언제든 끌 수 있어요)"
  → 동의 / 나중에 결정

[Step 3] 동의한 사용자에게만 — 2주 사용 후
  → "두뇌 건강 관리가 필요하다고 느끼시나요?
     데일리핏은 국내 치매안심센터와 연계하여 전문 상담을 연결해 드릴 수도 있어요."
  → 이 단계에서 비로소 치매 관련 기능임을 명시적으로 고지
```

## 6. MVP 범위 및 성공 지표

### MVP (4주 개발 목표)
- 발주·업무 메모 CRUD
- 포인트 적립 로직 (전환 기능은 파트너십 협의 후 추가)
- 1단계 텍스트 패턴 분석 (서버사이드 Claude API 활용)
- 온보딩 3단계 점진적 고지 플로우

### 성공 지표

| 지표 | 목표치 | 측정 방법 |
|---|---|---|
| DAU/MAU 비율 | 40% 이상 | 앱 로그 |
| 포인트 동기 유입 전환율 | 70% 이상 | 설치 후 설문 |
| 2단계 테스트 자발 참여율 | 20% 이상 | 이벤트 로그 |
| 치매안심센터 연계 클릭률 | 15% 이상 | 딥링크 클릭 수 |

## 7. 제외 항목 (Out of Scope — MVP)

- 다중 사용자 / 직원 계정 관리
- 스마트워치 연동 (MVP 이후 v2에서 적용)
- 음성 인식 기반 발주 입력 (텍스트 안정화 후 추가)
- 실시간 진단 결과 제공 (의료기기 규제 검토 후 적용)
